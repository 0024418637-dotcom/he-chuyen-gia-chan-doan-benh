# 🏥 Hệ Chuyên Gia Chẩn Đoán Bệnh

## 📌 Giới thiệu

Ứng dụng Java Swing mô phỏng hệ chuyên gia giúp chẩn đoán bệnh dựa trên triệu chứng người dùng.

## ⚙️ Chức năng chính

* 🧠 Chẩn đoán bệnh từ triệu chứng
* 📊 Thống kê bệnh và triệu chứng
* 🕒 Lưu lịch sử chẩn đoán
* 🔍 Tìm kiếm kết quả
* 📄 Xuất file kết quả

## 🛠️ Công nghệ sử dụng

* Java Swing
* Rule-based Expert System

## 🚀 Cách chạy

1. Mở project bằng NetBeans / IntelliJ
2. Run file `TrangChuForm.java`

## ⚠️ Lưu ý

Hệ thống chỉ mang tính chất tham khảo, không thay thế bác sĩ.
